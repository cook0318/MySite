SASS

SASS is a great way to write meaningful CSS quickly. It incorporates variables and functions to beautify HTML faster and more easily than plain CSS. Who doesn't love efficiency?

Bootstrap

Bootstrap is a CSS framework that provides a responsive and mobile-first design for modern websites. From FIFA to NASA, countless international companies use Bootstrap. So does this one.

Javascript

Javascript is the master behind the interactive-ness behind modern websites. From validation of forms, to dropdown menus, to pop-up alerts. You can thank Javascript.

jQuery

jQuery is a Javascript framework that simplifies object selection and document manuipulation. While potentially outdated, jQuery is still used by millions of websites.

Python

Python is a high-level programming language with endless usability. With variables, functions, objects, classes, and more, plenty of back-end applications are possible. It's also extremely readable and simple. Read the Zen of Python here.

Flask

Flask is a Python web framework. It allows us to connect HTML, CSS, Javascript, Python and more to make a completely functional website, backend included.
